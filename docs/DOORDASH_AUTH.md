# DoorDash Authentication Options

The DoorDash agent needs a way to log in. Auth methods (in order of preference):

| Method | 2FA | Setup |
|--------|-----|-------|
| **1Password** | Auto (TOTP from vault) | [1PASSWORD_SETUP.md](./1PASSWORD_SETUP.md) |
| **Profile sync** | Already logged in | [PROFILE_SYNC_GUIDE.md](./PROFILE_SYNC_GUIDE.md) |
| **Credentials** | Manual or Gmail auto | [2FA_OPTIONS.md](./2FA_OPTIONS.md) |

All options are supported. Use whichever works for you.
